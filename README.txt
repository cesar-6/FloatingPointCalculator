README – Floating-Point Calculator (Programming Assignment #2)
Author: Cesar Fuentes
=========================================================

This project implements a Floating-Point Calculator in MIPS.
It prompts the user for two real numbers and performs:

• Sum
• Difference
• Product
• Quotient
• Remainder

All functions:
• Receive parameters via the stack
• Return results in $f0
• Use floating-point instructions

Files:
• FloatingPointCalculator.asm — full implementation
• main.cpp — placeholder for rubric
• Calculator.h — placeholder header
• README.txt — documentation

Run Instructions:
1. Open MARS or QtSpim
2. Load FloatingPointCalculator.asm
3. Assemble and run
4. Enter two real numbers
5. View results for all 5 operations
